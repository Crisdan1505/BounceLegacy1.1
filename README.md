# App Citas + Calculator API

## Description

This project contains a .NET solution based on an appointment management system and an additional standalone Calculator API.

The Calculator API exposes REST endpoints for basic arithmetic operations such as addition, subtraction, multiplication, and division. A visual interface was also integrated into the web project to consume the API through HTTP requests.

## Technologies Used

- C#
- .NET
- ASP.NET Core Web API
- ASP.NET Core MVC
- Swagger / OpenAPI
- Git and GitHub
- HTML
- CSS
- Bootstrap
